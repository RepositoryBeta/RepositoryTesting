Tipos de sistemas operativos: Sistemas operativos para PC

Los más utilizados son el Windows, el Mac y el Linux

El uso de las ventanas es la insignia del sistema operativo de Microsoft, el Windows sistema estándar para ordenadores personales y de negocios.

El Mac Os ese sistema operativo de Apple Macintosh para línea de computadoras personales y estaciones de trabajo, la manzana es un logotipo.
